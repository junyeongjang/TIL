### 핵심 내용

- C에서 모든 변수는 사용되기 전에 선어해주어야 한다.

- \t는 tap을 의미한다.

- 표준 입출력

  - |      |                                                  |
    | :--: | :----------------------------------------------: |
    |  %d  |                      10진수                      |
    |  %o  |                      8진수                       |
    |  %x  |                      16진수                      |
    |  %c  |                      문자열                      |
    |  %f  |                 부동소수점 실수                  |
    | %6f  |   부동소수점 실수, 전체 길이만 여섯자리로 지정   |
    | %.2f | 부동소수점 실수, 소수점 이하 두 자리로 지정한다. |
